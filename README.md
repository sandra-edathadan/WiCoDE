# Interactive Event Website Design

An interactive, modern event website designed to deliver an engaging and visually appealing user experience with smooth animations and clean UI components.

This project is based on the original Figma design:
https://www.figma.com/design/rwRQJWfWt0W377qjC7rCVC/Interactive-Event-Website-Design

---

## Features

- Responsive and modern user interface
- Smooth animations and transitions
- Clean and reusable component-based architecture
- Performance-optimized frontend
- Developer-friendly project structure

---

## Tech Stack

- **Framework:** React / Next.js
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Package Manager:** npm

---

## Getting Started

Follow the steps below to run the project locally.

### Prerequisites

Make sure you have the following installed:

- Node.js (v16 or higher recommended)
- npm or yarn

---

### Installation


1. Clone the repository:
```bash git clone https://github.com/your-username/your-repo-name.git```

2. Navigate to the project directory:
```cd your-repo-name```

3. Install dependencies:
```npm install```

---

### Running the Development Server


Start the development server using:
```npm run dev```

The application will be available at:
```http://localhost:3000```
